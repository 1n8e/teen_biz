🛡 3️⃣ Защита от опасных людей (обязательно для защиты проекта)

Вот это нужно прямо выделить отдельным слайдом.

1. Чат только внутри сайта
   • Нельзя отправлять номер телефона
   • Нельзя писать адрес
   • Фильтр опасных слов

2. Система жалоб
   • Кнопка «Пожаловаться»
   • Быстрая блокировка аккаунта
   • Проверка модератором

3. Эскроу-система
   • Деньги замораживаются
   • Исполнитель получает их только после подтверждения

4. Рейтинг и отзывы
   • Низкий рейтинг = меньше заказов
   • Подозрительные аккаунты блокируются

5. Рекомендации для офлайн-встреч
   • Только публичные места
   • Желательно с родителями
   • Никогда не сообщать адрес

⚠ Важно: вы не просто говорите «мы защищаем», а объясняете КАК.
